Versioning

All public endpoints are versioned via the URL path. Breaking changes require a new major version — see Deprecation Policy before removing fields.

GET /api/v2/users/:id
Accept: application/json

Conventions

  • Use plural, lowercase resource names (/invoices)
  • Return ISO-8601 timestamps in UTC
  • Paginate with cursor, never offset
